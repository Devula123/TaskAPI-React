# Laravel TODO App (Inertia + React)

Prosta i nowoczesna aplikacja do zarządzania zadaniami, zbudowana w celu nauki nowoczesnego stacku technologicznego: **Laravel, Inertia.js oraz React**.

## Informacje

*   **Backend:** [Laravel 11](https://laravel.com) (PHP 8.2+)
*   **Frontend:** [React](https://reactjs.org) + [TypeScript](https://www.typescriptlang.org)
*   **Łącznik:** [Inertia.js](https://inertiajs.com) (Monolith feel, SPA experience)
*   **CSS:** [Tailwind CSS](https://tailwindcss.com)
*   **Środowisko:** [Laravel Sail](https://laravel.com/docs/11.x/sail) (Docker)

## ✨ Funkcje

*   ✅ **Pełny CRUD:** Dodawanie, wyświetlanie i usuwanie zadań.
*   🔄 **Real-time state:** Zmiana statusu zadania (wykonane/niewykonane) bez przeładowania strony.
*   🔐 **Autentykacja:** Pełny system logowania i rejestracji (Laravel Starter Kit).
